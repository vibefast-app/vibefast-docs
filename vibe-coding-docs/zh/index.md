# Vibe Coding Docs

[English](../en/index.md)

這一區提供公開的 Vibe Coding 學習內容，幫助讀者理解如何在 AI 協作條件下，用更快但仍然有邊界的方式建構產品。

Vibe Coding 不是把判斷交給模型，而是把模型納入你的工作系統，讓速度和清晰度一起提升。

## 你可以從這裡學到什麼

- Vibe Coding 的概念與邊界
- 常見術語與共同語言
- 安全工作方式
- prompting 的基本原則
- 工具與工作流如何配合
- 一位創作者/開發者的實際心得

## 與產品的關係

這一區講的是方法，而不是 private product repo 的實作細節。它可以幫助你更好地理解一種 AI 輔助建構方式，但不會變成私有產品的替代教學。
