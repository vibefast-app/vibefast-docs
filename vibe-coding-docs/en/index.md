# Vibe Coding Docs

[中文](../zh/index.md)

This section provides public learning material for Vibe Coding: a practical way of building with AI while still keeping human judgment, direction, and boundaries intact.

Vibe Coding is not about handing your product over to a model. It is about bringing models into your working system so speed and clarity improve together.

## What you can learn here

- what Vibe Coding is
- why it lowers the barrier to building products
- why Cloudflare is a strong platform for this workflow
