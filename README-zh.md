## 商業模板

跳過幾週的基礎設定，直接從一個跑起來的產品開始。

**內建功能**

- ✅ Auth — 註冊、登入、路由保護
- ✅ Admin 後台
- ✅ Blog CMS — Markdown 編輯器，每篇獨立 SEO 欄位
- ✅ Media Library — R2 圖片上傳和管理
- ✅ Stripe 收款 — 一次性付款，Webhook 預先設定好
- ✅ Resend 郵件通知
- ✅ Analytics — 內建，不需要第三方服務

**一個指令部署**

`npm run setup` 自動設定 Cloudflare Workers、D1、R2，並把你的 App 部署到全球。

**三個指令，全球部署**

```bash
git clone https://github.com/vibefast-app/vibefast.git my-app
cd my-app && npm install
npm run setup
```
